# ganti icon laucher 


### setingan pubspec yaml

```yaml
dev_dependencies: 
  flutter_launcher_icons: "^0.7.3"
  
flutter_icons:
  android: "launcher_icon" 
  ios: true
  image_path: "assets/icon/icon.png"
```


### jalankan diterminal

```terminal
flutter pub get
flutter pub run flutter_launcher_icons:main
```
